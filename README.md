# Сайт-визитка

Одностраничная визитка (landing) с анимациями. Один файл `index.html` — стили и разметка внутри.

## Просмотр

Открой `index.html` в браузере или разверни на любом хостинге статики (GitHub Pages, Netlify, Vercel и т.д.).

## Редактирование

Меняй текст, стили и разметку прямо в `index.html`.
