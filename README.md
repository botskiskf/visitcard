# Сайт-визитка

Одностраничная визитка с анимациями и AI-чатботом для консультаций (услуги, цены, контакты).

## Просмотр

Открой `index.html` в браузере или разверни на Vercel.

## Чат-бот

На сайте в правом нижнем углу — чат с консультантом (MiniMax API, скилл [cursor-ai-chatbot](https://github.com/evgyur/cursor-ai-chatbot)).

**Чтобы чат работал на Vercel:**

1. Получи API-ключ: [platform.minimax.io](https://platform.minimax.io).
2. В Vercel: проект → **Settings** → **Environment Variables** → добавь переменную **`MINIMAX_API_KEY`** (значение — твой ключ).
3. Передеплой проект.

**Локально:** API `/api/chat` не будет работать при открытии файла в браузере — нужен деплой на Vercel (или локальный сервер с этой же функцией).

## Данные для чата

- `knowledge.json` — база для ответов бота (услуги, цены, как связаться). Редактируй под себя.
- Имя консультанта и приветствие задаются в `index.html` в блоке `window.aiChatConfig`.

## Редактирование

Меняй текст и стили в `index.html`, данные для бота — в `knowledge.json`.
